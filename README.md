# docker-cme
Docker implementation for bleeding edge crackmapexec

Build

    ./build.sh

Run

    ./cme.sh

### Aliases
Or, alias the commands in aliases to your .bash_aliases (kali) or .bash_profile (osx) and launch with alias 'cme < options >'

    source /path/to/docker-cme/aliases
